# 🌐 CSE391 – Nền tảng phát triển Web

> 📘 *Môn học nền tảng giúp sinh viên tiếp cận và làm chủ các kiến thức cơ bản về phát triển Web hiện đại.*

---

## 🚀 Giới thiệu
**CSE391 – Nền tảng phát triển Web** cung cấp kiến thức nhập môn về phát triển website và ứng dụng web, tập trung vào **Frontend Web Development**.  
Môn học giúp sinh viên xây dựng nền tảng vững chắc để tiếp tục học các công nghệ web nâng cao.

---

## 🎯 Mục tiêu
✔ Hiểu kiến trúc tổng thể của một ứng dụng web  
✔ Xây dựng giao diện website cơ bản  
✔ Thiết kế website responsive  
✔ Làm quen với quy trình phát triển web và làm việc nhóm
